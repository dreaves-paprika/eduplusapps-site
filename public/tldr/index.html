Verify before installing
  1. Check the source repo — recent commits, open issues, README quality
  2. Check the publisher — official > known maintainer > random username
  3. Read tool descriptions — what permissions does it ask for?
  4. Test in a scratch project first; inspect via ToolSearch
  5. Pin versions: @x.y.z not latest
